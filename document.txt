‚úîÔ∏è FUNCTION 1 ‚Üí save_account(account_obj)
What it does

Opens database.txt

Adds a new line with the account data

Example stored line:

1001,John Doe,5000

Code:
def save_account(account_obj, filename="database.txt"):
    with open(filename, "a") as file:  # append mode
        line = f"{account_obj.account_number},{account_obj.name},{account_obj.balance}\n"
        file.write(line)
        print("Account saved successfully!")

‚úîÔ∏è FUNCTION 2 ‚Üí load_accounts()
What it does

Opens the file

Reads each line

Splits the values

Creates Create_Account objects for each line

Code:
from account import Create_Account

def load_accounts(filename="database.txt"):
    accounts = []
    try:
        with open(filename, "r") as file:
            for line in file:
                acc_no, name, balance = line.strip().split(",")
                account_obj = Create_Account(int(acc_no), name, int(balance))
                accounts.append(account_obj)
    except FileNotFoundError:
        print("No accounts found. File does not exist yet.")
    
    return accounts
